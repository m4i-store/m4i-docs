# Script Template: Introduction

Use this template as the baseline for every new M4I script documentation set.

Document these points:

- what the script does
- who should use it
- required dependencies
- which `m4i_bridge` capabilities it uses

## Mandatory integration statement

Scripts should integrate through `m4i_bridge` whenever bridge abstraction exists.

Direct framework/inventory integration should only be used if no bridge abstraction is available.
