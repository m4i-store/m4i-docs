# Support Policy

## Scope

M4I support is provided for:

- documented features in this hub
- supported provider combinations listed by each system
- current stable release paths

## Requirements for support requests

Include:

- resource versions
- relevant config snippets
- exact error messages
- trace IDs and debug snapshot output when available

## Documentation policy

Support decisions follow this hub as the source of truth.
Undocumented behavior should not be treated as stable contract.
